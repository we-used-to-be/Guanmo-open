# Guanmo v0.4.0

## ✨ 新增功能

- 新增 Markdown 图片插入能力，支持选择、拖拽、粘贴图片，并自动复制到笔记同级 `assets` 目录后插入相对路径。
- 新增 AI 自定义提示词配置，可在设置中调整助手回复风格。
- 增强 AI 上下文标签、RAG 检索与 Agent 工具调用体验。

## 🎨 UI优化

- 优化编辑区顶部栏、标签栏、右键菜单和侧边栏交互细节。
- 改进 Markdown 预览中的图片展示与放大查看体验。
- 优化多视图模式切换与命令面板入口。

## 🐛 Bug修复

- 修复图片插入时被文本文件扩展名白名单拦截，导致图片无法复制到 `assets` 并插入笔记的问题。
- 修复选区替换、Agent 工具解析和外部文件打开相关稳定性问题。

## ⚙️ 构建变化

- 版本号统一更新为 `0.4.0`。
- Windows 发布工作流继续通过 `v*` tag 自动构建 NSIS `.exe` 与 WiX `.msi` 安装包。

## ⚠️ 注意事项

- 本版本不内置 API Key，AI 与联网搜索能力仍需在设置中配置模型接口。
- 文件读写仅允许用户显式选择的工作区或文件；图片插入仅支持允许的图片扩展名。

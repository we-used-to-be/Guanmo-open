# Changelog

## v1.3.0 (2026-07-24)

### 🆕 新功能

- 启动时恢复上次会话的标签页与阅读位置，基于 Zustand 持久化 Store 实现跨会话状态延续
- 新增特性介绍弹窗，首次启动时以轮播形式展示软件核心功能，支持键盘与鼠标导航
- 设置页新增联网搜索 API 连接测试功能，可一键验证搜索服务可用性
- 新增预览内编辑快捷键：`Alt + 左键` 点击预览中的 Markdown 块即可进入源码编辑

### 🐛 修复

- 修复全屏分屏模式下左右区域边距逻辑，仅保留外侧边距避免内侧双重间距
- 修复 Embedding 连接测试用错端点以及 Tavily 认证方式不一致的问题

### ⚡ 优化

- 优化特性介绍弹窗图文比例与视觉布局，增加图片加载动画，无图页自动隐藏图片区域
- 更新检查间隔从 1 小时调整为 12 小时，减少不必要的网络请求

- 减少桌面端 JavaScript 体积与分块数量
- 清理敏感日志，并新增隐私日志静态检查

- 完善 Web 构建与体积检查

### 🔄 重构

- 统一文件重命名交互，改善提交、取消、失焦和失败恢复逻辑
- 优化跨模块状态调用方式，同时保留旧调用兼容

### 🔧 构建

- 搭建前后端基础自动化测试与质量检查（Vitest + ESLint）

## v1.2.3 (2026-07-18)

### 🆕 新功能

- 受限外部请求代理：新增 Rust 端受限 HTTP 代理，支持前端安全调用外部 API，解决了部分 token plan 请求的跨域问题
- 受限文件访问授权系统：实现文件访问授权机制，增强安全性与 UI 优化
- 文档改写流程：实现文档改写第一阶段功能
- 结构化文件总结：新增结构化文件总结功能
- 可信回答与本地 Web 研究：完善 AI 可信回答流程与本地 Web 研究功能
- 长期记忆优化：优化长期记忆触发与归纳策略，迁移至结构化长期记忆系统

### 🐛 修复

- 更新迁移工具链接、按钮样式规范与全屏控制条样式
- 改进设置页、选区解释与全屏控制体验
- 恢复自然解释并优化回答检索路由
- 修复标题选区与定向上下文读取问题
- 修复首次右键快捷提问需手动发送的问题

### 🔄 重构

- 数据库架构重构：经过深思熟虑，将 SQLite 确立为唯一正式数据库。当初为了兼容性临时采用 IndexedDB 作为兜底方案，导致双库并存的历史遗留问题。此次重构彻底抛弃这个兜底机制，统一为 SQLite，简化数据管理逻辑

### ⚡ 优化

- 优化 Markdown RAG 索引增量复用机制
- 优化大文档 Markdown 预览解析与渲染性能
- 数据库迁移策略：软件启动时会智能检测是否存在旧版 IndexedDB 数据，如发现则主动提醒用户是否需要迁移数据。同时提供独立的迁移工具，帮助用户将旧数据完整迁移到新数据库，确保平滑过渡
- 优化启动流程：新增统一任务调度器与单例 Promise 管理器，启动时只阻塞数据库初始化，UI 立即显示后分批闲时预热
- 向量库延迟加载：首次使用 RAG 时才按需加载，启动时不再阻塞
- AI 状态校验异步化：延迟执行，不阻塞编辑器使用
- 右键快捷提问自动发送：默认启用，无需手动点击发送

在此感谢 codex 的慷慨重置对本次大范围修复的支持～感谢 mimo 对迁移工具和首屏加载优化的大力支持～

## v1.2.0 (2026-07-13)

### 🆕 新功能

- 新增全屏模式，鼠标移至顶部唤起独立控制条，可快速切换编辑 / 预览 / Diff 视图，并通过“标签 / 文件”或 `Ctrl+B` 打开全屏文件侧边栏。全屏模式下 AI 助手改为小窗模式，拖动顶部调节位置，点击外部关闭弹窗，实现即用即走，有助于提升专注度。

- 新增 AI 吉祥物头像选项，可在设置页的 AI 头像入口中启用，AI 生成回复时会有“思考 - 写作”小动画。
- 新增模式闲时预热设置，可在设置页中开启，用于提升编辑 / 预览 / Diff 等模式切换体验。

### 🐛 修复

- 修复 Markdown PDF 导出时本地图片无法读取的问题。
- 修复 Markdown PDF 导出时普通段落换行丢失的问题。

### ⚡ 优化

- 优化界面视觉降噪，提升视觉专注度。
